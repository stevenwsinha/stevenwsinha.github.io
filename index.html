Top 5 Lister WEbaPP

View on Github

A single page web-app that enables users to create top 5 lists on any topic, and to share them with other users. Features include account creation and user authentication, viewing, liking, and commenting on other users lists, and the aggregation of all user lists into community lists of the most popular items.
